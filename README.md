## 1. Постановка задачи и требования

В рамках кейс-задачи требовалось разработать интерактивное консольно-браузерное приложение на языке JavaScript, выполняющее следующие функции:

1. Последовательный безопасный запрос у пользователя дня, месяца и года рождения.

2. Программное определение дня недели, соответствующего введенной дате.

3. Математическая проверка года рождения на статус високосного.

4. Точный расчет текущего возраста пользователя с учетом дня и месяца рождения в текущем году.

5. Стилизованный посимвольный вывод даты рождения в консоль с помощью матричных символов-звёздочек (*), имитирующих электронное табло.

## 2. Архитектура проекта и модульная структура

Программа спроектирована с использованием современной модульной системы ES Modules (ESM). Вместо написания всего кода в одном файле, логика была декомпозирована на изолированные утилиты в папке utils/. Это существенно повышает читаемость, тестируемость и сопровождаемость программного продукта.

Структура файлов проекта:

* index.html — Входная точка приложения, подключающая управляющий скрипт как модуль (type="module").
index.js — Главный координирующий модуль, содержащий бизнес-логику вычислений и вывод результатов пользователю.
* utils/inputDate.js — Логика последовательного опроса пользователя и валидации сырых данных.
* utils/isValidDate.js — Математические алгоритмы валидации календарной сетки и вычисления високосного года.
* utils/getUserYearOld.js — Функция фабрикации/валидации результирующего объекта даты.
* utils/printLargeNumber.js — Алгоритм генерации псевдографического вывода «электронного табло».
